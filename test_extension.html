üéØ Get Started in Minutes

Setting up NativeFoundationModels is quick and straightforward. Follow these steps to integrate native AI into your web applications.

  1. Load the NativeFoundationModels extension in Chrome: navigate to chrome://extensions/
  2. Enable Developer mode and click Load unpacked
  3. Select the chrome-extension folder from the NativeFoundationModels project
  4. Copy the generated Extension ID and run ./update_manifest_with_id.sh <ID>
  5. Refresh your web application and start using window.nativeFoundationModels

That's it! You now have access to powerful language models directly in your web applications with complete privacy and zero latency.
